# ToDo Manager

A simple command-line ToDo Manager built with Python.

## Features

* Add new tasks
* Delete tasks
* View all tasks
* Mark tasks as completed
* Save tasks to a CSV file
* Load tasks automatically from a CSV file

## Project Structure

```text
Python-ToDo-Manager/
│
├── main.py
├── data/
│   └── tasks.csv
├── models/
│   └── task.py
└── services/
    └── todo_list.py
```

## Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
* CSV File Handling

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/Desmond-coder/Python-ToDo-Manager.git
```

2. Move into the project directory:

```bash
cd Python-ToDo-Manager
```

3. Run the application:

```bash
python main.py
```
